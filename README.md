# Business Requirements Analyst Skill

A professional Claude AI Skill designed to transform vague business ideas into structured, development-ready documentation using industry-standard methodologies (BABOK, Design Thinking, Lean Startup) and AI-optimized specifications (EARS, ADR, SDD).

---

## üöÄ Main Features

- **Smart Discovery**: Automatically analyzes provided documents, images, and context to avoid redundant questions.
- **Process Kickoff**: Clear 6-phase roadmap presentation at the start of every project.
- **EARS Syntax**: Functional requirements written in "Easy Approach to Requirements Syntax" to eliminate ambiguity.
- **Auto-ERD**: Automatic extraction of entities and generation of visual diagrams using Mermaid.js.
- **Traceability Matrix**: Full audit trail connecting business goals to specific code implementation tags.
- **ADR Management**: Capture and document critical technical decisions (Architecture Decision Records).
- **Quality Gates**: Automatic maturity scoring to ensure documentation is ready for development.
